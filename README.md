# ChillDuanHeiSong
基于思源宋体（SourceHanSerifSC）修改，横向加粗，字体横竖粗细相差更小，结合了黑体与宋体的特征，既有黑体粗壮的特性，又有宋体优雅的饰角，比黑体更雅致，比宋体更硬朗。介于黑体和宋体之间的基础字体风格。名为端黑宋。<br>
